# GTField

In order to support user so that they can using Smartphone for work effectively, we created an app called GTField with the following functions:

Maps View:
+ Displays user's location on the map in real-time
+ Displays worlwide roadmap, satellite and terrain map
+ Displays geospatial data from GeoServer's geodatabase
+ Displays offline map
+ Displays all waypoints, tracks on the map
+ Displays all geotagged photos photos on the map

Map Tools:
+ Map
+ Search natural places, and addresses
+ Search local waypoints
+ Download geospatial data from GeoServer's geodatabase
+ Record waypoints, tracks
+ Export the latitude, longitude, and altitude to csv file by iTunes, and email
+ 

Camera:
+ Capture geotagged photos (using digital camera and GPS with built in to get GPS information at the time the photo is taken to generate the geotagged photos)

Connect and access resources (connect to the server and access resources using the app):
+ Input the URL of the GeoServer layers
+ Access and list all layers from the GeoServer
+ Select layers for display geospatial data
+ Select layer to access specific feature informations
